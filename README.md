# redirect-gg
tera-proxy module to redirect spawn location in TERA dungeon `Ghillieglade`

## Dependency
- `command` module
- `tera-game-state` module

## Usage
- __`gg`__
  - Toggle on/off
  - Default is `true`

## Config
- __`enable`__
  - Initialize module on/off
  - Default is `true`
