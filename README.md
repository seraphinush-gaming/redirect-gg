# redirect-gg
tera-proxy module to redirect spawn location in TERA dungeon `Ghillieglade`

## Dependency
- `Command` module

## Usage
- __`gg` · `ㄱㄹ`__
  - Toggle on/off
  - Default is `true`

## Config
- __`enable`
  - Initialize module on/off
  - Default is `true`
